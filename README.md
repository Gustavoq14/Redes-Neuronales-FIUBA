# Redes-Neuronales-FIUBA
El siguiente repositorio contiene los Trabajos Prácticos (TP) realizados durante el curso de grado y posgrado de [Redes Neuronales FIUBA](https://campusgrado.fi.uba.ar/course/index.php?categoryid=438) (Buenos Aires-Argentina). Espero sea de ayuda para todos aquellos que se estan iniciando en el mundo de la IA (Inteligencia Artificial).

El curso emplea como bibliografía principal el libro [Introduction To The Theory Of Neural Computation-Westview Press, CRC (1991,2018)](https://nessie.ilab.sztaki.hu/~kornai/2020/AdvancedMachineLearning/Hertz_Krogh_Palmer__IntroToNeuralComp.pdf). 

Cada uno de los trabajos prácticos cubren los siguientes temas:
- TP1: Red de Hopfield y Modelo de Ising.
- TP2: Aprendizaje Supervisado (Perceptron Simple y Multicapa).
- TP3: Aprendizaje No-Supervisado (Red de Kohonen).
